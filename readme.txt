== Description ==
Author: [Web Design London](http://www.smallbizwebdesigns.co.uk "Web Design London")

Plugin Link: http://www.smallbizwebdesigns.co.uk/wordpress-125x125-ad-widget-plugin.html

It will display 125 x 125 sponsor's ad (maximum 6 ads) in your blog/website sidebar.


**Plugin Features**

* Add 125x125 size advertisement.
* Use widgets to put ads on your site
* Responsive Layouts
* Put random ads


== Installation ==

1. Download the plugin and navigate to wp-contents/plugins and extract there.
2. Go to wp-admin/plugins and Activate LWD 125x125 Ad Widget plugin.
4. Navigate to  Appearance > Widgets, drag the LWD 125x125 Ad Widget to wherever you want it to be and click on Save.
